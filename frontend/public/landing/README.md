# Landing Page Images

Drop the following image files here for the landing page to render properly:

- `hero-field.jpg` — large hero/about backdrop (use the sunrise cornfield image, ~1024x1024 or larger)
- `wheat-field.jpg` — secondary section accent (use the wheat-with-blue-sky image)
- `corn-walk.jpg` — about section side image (use the person-walking-cornrow image)

If a file is missing, Next.js will return 404 and a soft gradient fallback will show.
